# Home Library Service

## Downloading

```
git clone {repository URL}
```

## Installing NPM modules

```
npm install
```

## Running application

```
npm start
```

## Testing

!!! BEFORE STARTING TESTS !!!

1. run:

```
npm run seed
```

2. Start the server

```
npm start
```

3. Run tests in a separate terminal

```
npm run test
```
